About Me

Saya adalah mahasiswa Rekayasa Perangkat Lunak di Politeknik Manufaktur Negeri Bangka Belitung dengan minat besar pada pengembangan aplikasi web dan mobile serta UI/UX design. Selama perkuliahan, saya telah mengembangkan berbagai proyek seperti aplikasi mobile berbasis Flutter, sistem berbasis web menggunakan PHP dan Laravel, serta desain antarmuka aplikasi menggunakan Figma. Beberapa aplikasi yang saya bangun juga terintegrasi dengan Firebase sebagai backend dan real-time database untuk mendukung sinkronisasi data secara langsung. Saya menikmati proses mengubah ide menjadi produk digital yang nyata dan bermanfaat. Saya terbiasa menggunakan teknologi seperti Flutter, HTML, CSS, JavaScript, PHP, Laravel, MySQL, Firebase, serta tools seperti Git, GitHub, dan Figma dalam pengembangan proyek. Saat ini saya terus belajar dan terbuka untuk kolaborasi proyek, magang, maupun kesempatan di bidang frontend development dan web development.
